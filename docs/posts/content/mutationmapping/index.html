Overview

This page compiles reference structures, strains, numbering schema anchors, and antigenic site definitions for influenza hemagglutinin (HA) and neuraminidase (NA). The goal is to provide a standardized reference page that is easy to use for annotation.

For each protein, the tables below report:

  • Type/subtype
  • PDB reference structure
  • Reference strain
  • Starting motif or head-domain start
  • Antigenic site name
  • Epitope type
  • Residue ranges or positions
  • Reference source
